# Project 05: Sequence Detector using Mealy and Moore FSM

## Objective

This project implements a 1011 sequence detector using both Mealy and Moore finite state machines.

## Key Concepts

- Finite State Machine
- State register
- Next-state logic
- Output logic
- Mealy FSM
- Moore FSM
- Overlapping sequence detection
